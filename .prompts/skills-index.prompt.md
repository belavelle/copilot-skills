# Skills Index

You have access to a small set of Copilot skills defined in this repository.
Each skill is a self-contained package with its own instructions and resources.

Use only the **name and description** below to reason about applicability.
Do NOT load or summarize any SKILL.md unless explicitly instructed.

---

## Available Skills

### skill-creator
Create or update Copilot skills that encode reusable workflows, domain knowledge,
or helper resources. Use when designing a new skill or refining an existing one.

### frontend-design
Create distinctive, production-grade frontend interfaces with strong visual
direction. Use when building or styling web components, pages, dashboards,
landing pages, or frontend apps. Avoid generic or template-like UI.

---

## How to Use This Index

1. Ask the user what they are trying to accomplish.
2. Identify the most relevant skill (or none).
3. Explain **why** the skill applies.
4. Ask for confirmation before proceeding.

If no skill applies, explain why and continue without a skill.
